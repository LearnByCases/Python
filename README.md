# Python
Python编程范例。


## jupyterlab环境安装方法
```
pip3 install jupyterlab
pip3 install jupyterlab-language-pack-zh-CN

# 启动jupyterlab
jupyter-lab  --allow-root --autoreload --ip=127.0.0.1 --port=8080
```
* Windows系统中建议使用wsl环境
* Python3.6+
